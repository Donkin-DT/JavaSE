######流程控制结构

```java
理解：
	1.程序执行顺序可以是顺序的、可以按条件进行选择的、可以是不断循环的
	2.一个程序中，一般来讲不仅仅是一个单独的结构，往往是几种结构的组合

分类：😀
	1.分支结构：
		理解：
			程序从两条或多条语句中选择一条执行的结构，又称选择结构或条件结构
			
		单分支结构：
			语法：
				if(条件){
					语句
				}
			
			注意：😀
				① 条件的结果必须是 boolean 类型
					形式：
						关系表达式，如 a>9 或 a==10
						逻辑表达式，如 a>6 && b<10
						boolean 变量，如 flag
						boolean 常量，如 true|false
						boolean 变量赋值语句，如 flag = true
                
				② 大括号中语句可以是任意命令语句（输出、输入、变量、运算等）,且可为 0 条或多条
				③ 大括号中只有一条语句，可以省略大括号
			
			执行顺序:
				先判断条件是否成立，成立则执行大括号中语句，反之跳过大括号执行 if 结构后语句

	2.双分支结构：
		语法：
			if(条件){
				//代码块 1
			}else{
				//代码块 2
			}
		
		注意：
			① 条件同单分支条件语法
			② if 和 else 大括号都可省略，默认执行第一条语句
		
		执行顺序：
			条件成立则执行代码块 1，反之执行代码块 2		

	3.多分子结构：
    	语法：
    		if(条件1){
				//代码块 1
			}else if(条件2){
				//代码块 2
			}
			....
			else{
				//代码块 n
			}

		注意:
			1.与单分支、双分支条件和代码块语法一样
			2.else 语句块可省略
			3.else 不能开头
			4.多个 else if 语句块顺序可以调换，但影响效果
			5.依次从上往下判断条件，如果上面条件已经成立，下面条件将不再判断
			6.多重 if 一定要和多个 if 结构区分开
		
		执行顺序：
			先判断条件 1，成立则执行代码块 1;否则继续判断条件 2，成立则执行代码块 2;... ;如果都不成立，则执行 else 后代码块 n

	4.嵌套分支结构：
		理解：
			1.一个分支结构中又完整嵌套了另一个完整分支结构
			2.单分支、双分支和多重分支都可充当外层分支或内层分支
		
		语法：
			if(){
				//代码块 1
				if(){
					//代码块 2
				}else if(){
					//代码块 3
				}else{
					//代码块 4
				}
			}

	5.switch结构 😀
		语法：
			switch(变量/表达式){
				case 常量1：语句 1；break;
				case 常量2：语句 2；break;
				case 常量3：语句 3；break;
					...
				default:语句 n;
			}
		
		注意：😀
			① switch 中变量或表达式类型只能是 int 或能自动转换成 int 类型（byte、short、char）,或 String (jdk1.7)，或 enum (jdk1.5)
			② case 后面只能是常量值
			③ case 后面常量值不能重复
			④ case 后语句可以是0条或多条
			⑤ break 可以省略，但有可能影响效果
			⑥ default 语句块可以省略，位置可以提到 case 语句块前面，可能影响效果
			⑦ switch 结构适合处理等值判断
		
		执行顺序：
			先判断变量值，然后依次从上往下匹配各 case 后常量值，如果匹配成功，则执行对应 case 后语句，直到遇到 break 为止（如没有 break，则执行到最后）；如果都不匹配，则执行 default 语句（如果没有default，则不执行）
	
if 与 switch 对比：😀
	相同点：
		都可以实现分支结构
	
	不同点：
		switch 结构处理的分支，可以用 if 结构
		if 结构处理的分支，不一定能用 switch 结构；必须满足以下条件：
			① 等值判断
			② 判断的类型必须是int、char、byte、short、String、枚举

	注意：如果可以用 switch 结构处理，最好用 switch 结构，因为效率较高,而且语义性更强
```
