###### 变量😀

```java
引入：
	程序中使用的数据存储在内存中
	
计算机存储设备：
	1.永久性存储：硬盘、磁盘、u盘
	2.临时性存储：内存（容易丢失）
	变量相当于内存中一个数据存储空间的表示 
	
三要素：
	1.数据类型
	2.变量名
	3.变量值
    
特点：
	1.变量把值存储在内存中，特点：容易丢失
	2.变量在一个作用域内不能重名
	3.变量需要先声明、赋值、最后再使用
	4.变量声明和赋值可以合二为一
	5.变量在同一个类型范围内，值可以更新
	
使用步骤：
	1.声明变量：
		int num；
		double score；
		char gender；
		String name；
		
	2.赋值：
		num = 99；
		score = 1.5；
		gender = ‘\n’；
		name = “donkin”；
		
	3.使用：
		System.out.println(num);
		System.out.println(score);
		System.out.println(gender);
		System.out.println(name);
```

###### 数据类型 😀

```java
分类：
	1.基本数据类型：😀
		整型：
			byte short int long
		
		浮点型：
			float double
		
		字符型：
			char
		
		布尔型：
			boolean
		
	2.引用数据类型：
		使用class定义；比如：
			String
		使用interface定义；比如：
			List
		数组类型
	
基本数据类型介绍：😀
	理解：
		计算机存储单位：字节和位
		1个字节 = 8位
		计算机底层的存储形式为二进制
		10 ——> 0000 1010
		0000 0000 ~ 1111 1111
    
	1.整型：😀
		byte 字节整型      1个字节   -128 ~ 127（-2^7 ~ 2^7-1）
		short 短整型      	2个字节   -2^15 ~ 2^15-1
		int 整型 	         4个字节	-2^32 ~ 2^31-1
		long 长整型 	 	8个字节   -2^63 ~ 2^63-1
    
		0000 0000 ~1111 1111
		0000 0000 0000 0000~1111 1111 1111 1111
    
		特点：
			1.只能存储能表示的数值范围内的数，否则会报不兼容错误
			2.编译器默认的整型为 int 类型
			3.long 类型的常量值要用 L 或 l 标记
			
	2.浮点型：😀
		float 单精度浮点型	4个字节
		double 双精度浮点型   8个字节
		
		特点:
			1.浮点型的存储和整型的存储不一样
			2.浮点数：
				符号位 + 指数位 + 尾数位
				代表：2.2
			3.如果想保存更高精度数，可以使用 BigDecimal 类，或可以直接使用格式化方法保留小数点后几位
			4.编译器默认的浮点型为 double 类型
			5.float类型的常量值，要用 f 或 F 标记
			
		浮点型的常量的表示形式：
			1.5
			1.5f
			.12345
			2.3E12 科学计数法
			
	3.字符型:😀
		char 字符型 2个字节 使用 unicode 字符集
		常见编码表:
			ASCII 码表
				缺点:
					只能包含 128 个字符
				
			unicode 码表
				优点：
					包含了世界上所有的符号，不会出现乱码问题
				缺点：
					固定长度的编码格式，每个字符占2字节，空间有些浪费
					
			gbk
				优点：
					包含中文符号
				缺点：
					没有包含世界上的所有符号
					
			utf-8 
				在 unicode 码表上的继续实现
				优点：
					包含了世界上所有的符号，不会出现乱码问题，可变长编码格式，每个字符根据需要可以占1~6个字节
					
		特点：
			1.每个 char 类型变量存储时，都需要先去对应的码表找到对应的码值，然后存储
				char ——> 码值 ——> 存储（char类型的底层实际上就是以整型存储的）
			2.每个 char 类型的变量读取时，都需要先将码值根据码值找到对应的字符，然后访问
				码值——>char ——>读取
				
		char 类型的常量值的形式:
			char c = 'a'；
			char c = 100；
			char c = '\u0002'；
			
	4.布尔型:😀
		boolean 布尔型 1个字节 true / false
                
		特点：
			1.一般保存关系结果，用于条件判断或循环中
			2.数据类型的转换
			
基本数据类型之间的转换:😀
	大小规则：byte——> short ——> int——>long ——> float ——> double
	char 和 byte，short 之间不能转换
	char ——> int ——> long ——>float ——>double
	
	自动类型转换：😀
		规则：
			小 ——> 大
		
	强制类型转换：😀
		规则：
			大 ——> 小
			
		语法：
			类型 变量名 = （类型） 变量值；
			
	特点：😀
		1.自动提升原则：
			表达式结果类型为操作数类型最大的
		2.强转符只针对最近的操作数有效，所以一般需要通过小括号提升优先级
		3.char 类型
			可以直接保存 int 常量值，但不能保存 int 变量值；
			在进行运算或比较时，都当作 int 类型处理
		4.byte 和 short 类型:
			在进行运算时，都当作int类型处理
		
String 类型和基本类型转换:
	说明：针对8大基本数据类型（包括boolean）
		基本数据类型 ——> String类型
		
	语法：
		拼接空字符串即可
		int i = 100；
		String s = i + "";

String 类型 ——> 基本数据类型:
	语法：
		使用基本数据类型包装类调用 parseXX 方法即可
		String s = "12"；
		int i = Integer.parseInt(s);
		要求：值为有效的 XX(int...) 类型
```

###### 标识符😀

```java
概念：
	java中凡是可以自己起名字的地方，都称为标识符
	
比如：
	变量 方法 类 包 接口等
	
命名规则：😀
	只能包含字母、数字、下划线 和 美元符号（$），不能是关键字或保留字，不能以数字开头
	
命名规范：😀
	1.遵循见名知义
	2.变量名或方法名：遵循驼峰命名法 例如：myJavaScore
	3.类或接口名：遵循 Pascal 命名法（即：首字母大写，后面遵循驼峰命名法）
```

###### 运算符😀

```java
算术运算符：
	+ - * /
    
	特点：
		/ 左右两边的操作数如果都是整型，则结果取整型
		a % b = a -(int) a / b * b
		13.5%3=13.5- 13/3*3=1.5
		-13%-3= -13 - (-13)/3*3 = -1
    
	运算顺序：
		从左至右
		
自增/自减算术运算符：😀
	++ --
    
	特点：
		i++；
		--i；
		++i；
		i--；
		int j = i++；
		1.既可以作为独立语句使用，又可以作为表达式使用
		2.如果作为独立语句使用，则 i++; 等价于 ++i；等价于 i = i + 1;
		3.如果作为表达式使用，则
			int j = i++；先赋值后自增
			int j = ++i；先自增后赋值
			自增、自减运算符，编译器做了内部的优化，会进行自动的类型转换，而且效率也提高了。即：byte b = 1；b++；不会报编译错误
			
关系运算符：
	> < >= <= == !=

	特点：
		1.关系表达式的结果类型肯定是 boolean 类型
		2.关系表达式一般当作条件放在 if 结构或循环结构中
		
逻辑运算符：
	引入：
		如果令狐冲 java 成绩>90,则岳灵珊就和他在一起
			if(javaScore>90){
			}
			
		进阶：如果令狐冲 java 成绩>90并且 music > 85 并且  dance == 100 或  html > 70 则岳灵珊就和他在一起 if(java>90&&music>85&&dance==100||html>70){}

	作用：
		为了连接多个关系表达式的，最终的结果依然是boolean类型
		
	摘要：
		&& 短路与 条件一 && 条件2 两个条件都成立，结果为 true
		|| 短路或 条件1 || 条件2 只要有一个条件成立，结果为 true
		！ 非  !条件 如果条件本身成立，结果为 false
		& 逻辑与 条件1 & 条件2 两个条件都成立，结果为 true
		| 逻辑或 条件1 | 条件 只要有一个条件成立，结果为 ture

	&& 与 & 对比：
		&& 如果第一个条件不成立，则不再判断第二个条件，直接结果为 false
		& 不管第一个条件是否成立，都要判断第二个条件
		
	|| 与 | 对比：
		|| 如果第一个条件成立，则不再判断第二个条件，直接结果为 true
		| 不管第一个条件是否成立，都要判断第二个条件
		
赋值运算符：
	摘要:
		1- 1基本赋值运算符:
			= 
        
		1-2复合赋值运算符:	
			+= -= %= *= /= <<= >>= ~= 等
		
	特点:
		1.运算顺序从右往左
		2.赋值号左边只能是变量
		3.赋值号右边可以是常量值，变量、表达式
		
	复合赋值运算符：
		i += 3；等价于：i =  i + 3；
		底层进行了类型转换，编译器进行了内部优化
		使用复合赋值运算符的变量必须进行初始化
		
条件运算符（三目运算符）:😀
	摘要:
		条件 ? 操作数 1 ： 操作数 2
		
	特点:
		先判断条件是否成立，如果成立，则最终结果为操作数 1 的结果，否则为操作数 2 的结果
		
运算符优先级顺序: 😀
	1 ()
	2 ！
	3 ++ --
	
	4 基本算术运算符
	位运算符
	5 关系运算符
	6 逻辑运算符
	7 三目运算符
	8 赋值运算符
```
